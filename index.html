如何获取 Firebase 配置？
步骤：
1. 前往 Firebase 控制台 登录 Google 账号
2. 创建项目，添加 Web 应用
3. 复制配置 JSON（含 apiKey / databaseURL）
4. 左侧菜单 → 构建 → Realtime Database → 创建数据库
5. 选"测试模式启动" → 启用
6. 粘贴配置到上方文本框 → 保存配置
注意：测试模式允许任何人读写，适合内部使用。
